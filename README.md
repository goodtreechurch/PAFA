# YOUTH OF PAFA — 프로젝트 사이트

목회자 가정(**Pa**stor **Fa**mily)의 자녀를 섬기는 여름 수련회 프로젝트 페이지입니다.
**좋은나무교회**가 주최하고, 다섯 참여 교회(가락로고스·토브·샘솟는약수·윌로우·좋은나무교회)가 함께하며,
이번 **VOL.1 (2026 · 양양)** 을 시작으로 매년 이어갈 수 있도록 구성했습니다.

## 구성

```
pafa/
├── index.html        # 홈 — 포스터형 히어로, 메시지, 영상, PAFA 소개, 연합 교회, CTA
├── about.html        # 소개 — PAFA의 의미, BE BRAVE 메시지, 첫 번째 여정
├── info.html         # 안내 — 일시·장소·대상·연합 교회 등 핵심 정보
├── README.md
└── assets/
    ├── css/style.css
    ├── js/main.js
    ├── pafa-2026-promo.mp4   # 홍보 영상
    └── img/
        ├── poster.jpg        # 공식 포스터 (홈 '공식 포스터' 섹션 + OG 이미지)
        ├── banner.jpg        # 가로형 배너 (홈 하단 스트립)
        ├── wave-circles.jpg  # 히어로 겹친 원 안의 파도 (CSS 마스크로 원형 처리)
        ├── wave.jpg          # 슬로건 밴드 배경의 파도
        └── video-poster.jpg  # 홍보 영상 썸네일
```

> 모든 파도/포스터 이미지는 보내주신 **공식 포스터(IMG_6347)** 와 **배너** 원본에서 추출했습니다.
> 포스터가 인쇄용 CMYK라 웹 표시용 sRGB로 변환해 사용했습니다.
> (함께 올려주셨던 001·002 png는 채팅 미리보기 캡처 스크린샷이라 웹용으로는 제외했습니다.)

## 디자인 — 공식 포스터 기준

- **컬러**: 스카이 블루 `#33bdee` + 파스텔 옐로우 `#f3efa1` + 화이트.
- **타이포**: 디스플레이 **Archivo Black**, 본문 **Pretendard**, 손글씨 악센트 **Caveat**.
- **텍스처**: 포스터의 그레인 느낌을 SVG 노이즈 오버레이(`.grain`)로 재현.
- **모티프**: 포스터의 겹친 두 원(파도) 을 히어로에 `.circles` 로 구현.
- **슬로건**: *Catch the Wave, Be Brave!* / 파도를 잡으라, 담대하라.

## 포스터의 파도 이미지를 그대로 쓰려면

히어로 원형 안에는 현재 영상에서 추출한 `assets/img/wave.jpg`(잔잔한 바다)가 들어가 있습니다.
포스터의 역동적인 파도 사진을 그대로 쓰려면, 그 이미지를 **`assets/img/wave.jpg` 같은 이름으로 덮어쓰기**만 하면 코드 수정 없이 반영됩니다.
(가로로 긴 형태가 두 원에 가장 자연스럽게 담깁니다.)

## 보기 / 업데이트

`index.html` 을 브라우저에서 열면 됩니다. 별도 빌드/서버가 필요 없는 정적 사이트입니다.
폰트는 CDN으로 불러오므로 인터넷 연결 시 디자인이 완전하게 표시됩니다.

다음 시즌(VOL.2~) 업데이트: 로고 `VOL.1` 배지와 날짜/장소 값을 교체하고,
`about.html` 의 타임라인(`.journey`)에 지난 시즌을 한 줄씩 추가하면 됩니다.
